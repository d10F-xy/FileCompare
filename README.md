# (C# 코딩) File Compare

## 개요

- C# 프로그래밍 학습

- 1줄 소개: 사용자가 메뉴를 선택하면 주문 목록과 총 금액을 계산해주는 프로그램

- 사용한 플랫폼: C#, .NET Windows Forms, Visual Studio, GitHub

- 사용한 컨트롤: Label, RadioButton, CheckBox, ListBox, Button

- 사용한 기술과 구현한 기능:

  - Visual Studio를 이용하여 UI 디자인
  - 조건문(if, else if)을 활용한 메뉴 선택 처리
  - ListBox를 이용한 주문 목록 출력
  - 변수를 이용한 총 금액 누적 계산 기능 구현
  - CheckBox와 RadioButton을 활용한 단일/복수 선택 기능 구현
  - 버튼 클릭 이벤트를 통해 주문 추가 및 초기화 기능 구현
